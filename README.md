angular_sudoku
==============

Sudoku game built with angular.js

## Install
```
npm install
```
## Build
```
gulp
```

[Visit localhost:8080 to play!](http://localhost:8080)

Or, if you prefer using Vagrant (required plugin hostmanager):

## Run
```
vagrant up
```

[Visit sudoku.local to play!](http://sudoku.local)
